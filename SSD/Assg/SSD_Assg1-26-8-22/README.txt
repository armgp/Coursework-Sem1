q1
------------------------------------------------------------------------------------
 1. A promt is given saying "Enter a number greater tha 0".
 2. Input should be a number greater than 0.
 3. Output will be printed. There will be a heading and below it
    each set of twin primes are printed along with its product and digital root. 
    Each set is numbered from 1 and is divided by white lines, an example is given below.

|-----------------------------
|1.
|Prime1: 3, Prime2: 5
|Prime1*Prime2 = 15
|Digital Root = 6
|-----------------------------

 4. Finally at the end of the input the Sum of all Digital Roots is printed.
 5. For large input values a message will be shown "Kindly wait for few seconds".

q3
------------------------------------------------------------------------------------
 1. A prompt is printed asking to enter a 4 digit number.
 2. If the entered number is not 4 digits or if its 1111 or 0000 the promt
    again asks for a 4 digit number.
 3. The output of a 4 digit number is the Kaprekars routine executed and
    each number starting from the input number to Kaprekars constant is printed.


q4
-------------------------------------------------------------------------------------
 1. A promt to enter the year is given
 2. A proper year is consodered as any year from 0 to 9999
 3. Any year not in this range if entered another prompt asks the
    user to input the year lesser than 100000
 4. If no palindrome years are available - 
   "No Palindrome days available in the given year" is printed
